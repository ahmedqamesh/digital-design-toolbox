# FPGA Digital Logic Labs (Basys3)

This repository contains a set of introductory FPGA laboratory exercises implemented in **VHDL** and targeted for the **Digilent Basys3 FPGA board**.  
Each lab demonstrates a fundamental concept of **combinational digital logic** using switches as inputs and LEDs or seven-segment displays as outputs.

---

## Target Platform

- **Board:** Digilent Basys3
- **FPGA:** Xilinx Artix-7 (XC7A35T)
- **Toolchain:** AMD/Xilinx Vivado
- **Language:** VHDL (IEEE 1076)
